The library is static by design. Durable play state, moderation, and generated assets live with the orchestrator.

Live Shard lonelyforest.com runs the current V2 world, wallet sessions, actions, SSE stream, and room state.
Library Site lonelyforestlibrary.com keeps a fast public-facing archive surface on S3 and CloudFront.
Persistent Store The AWS service mounts EFS at /data for snapshots, SQLite events, and generated media.

This page is the first shelf. As the worldpack grows, the library can publish generated card manifests, room catalogs, release notes, and curated transcripts without coupling the public archive to the live game server.
